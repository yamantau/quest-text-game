# MagicQuest

- Junit

## Usage

- Вам потребуется Tomcat 9 и соответсвующие ему импорты HttpServlets
- Tomcat Configure: 
  - URL: http://localhost:8080/start
  - Application context: /

## Tests

Проект покрыт юнит-тестами JUnit. Для их запуска выполните команду:
```sh
mvn test
```
